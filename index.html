newTheme()

You can define a new theme using this code on your script.js

 
                        colors = {
                            Name : 'fullWhite',
                            Primary : '#FFFFFF',
                            Light: '#FFFFFF',
                            Secondary: '#FFFFFF',
                            Background: '#FFFFFF',
                            RippleEffect : '#FFFFFF'
                        }
                            newTheme(colors);

setTheme()

Updating the whole page theme using setTheme() method: 

setTheme("fullWhite");
